Troop Leading Procedures

Troop leading procedures are a set of guidelines for leaders to follow when leading troops in combat.

  • Receive the Mission
  • Issue a Warning Order
  • Make a Tentative Plan
  • Start Necessary Movement
  • Conduct Reconnaissance
  • Complete the Plan
  • Issue the Order
  • Supervise and Refine the Plan

OCOKA

OCOKA is a mnemonic device used to remember the key elements of a combat order.

  • O - Orientation
  • C - Course of Action
  • O - Obstacles
  • K - Key Terrain
  • A - Avenues of Approach
